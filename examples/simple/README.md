# Simple swmmx examples

These are deliberately flat learning scripts: no `main()` function, no `try` blocks, and no shared helper imports. Run them from the repository root just like the fuller examples:

```bash
python examples/simple/01_open_validate_run.py
```

They mirror the ten richer examples one-for-one, but trade defensive handling for a shorter reading path. The export example keeps GIS and Excel lines commented because those require optional dependencies.
