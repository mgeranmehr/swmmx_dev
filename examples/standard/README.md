# Standard swmmx examples

These are the fuller runnable demonstrations of the public `swmmx` API. They mirror the ten simple learning scripts in the parent `examples/` folder, but keep helper functions, safer branching, and more explanatory handling around optional features.

Run them from the repository root:

```bash
python examples/standard/01_open_validate_run.py
```

They still use `examples/example.inp`, never overwrite it, and write generated files into `examples/output/`.
